HotSpot

Winner at Garuda Hacks 2020

SMS Assist

Finalist at McHacks 2020

Realtime SMS AI designed to handle commercial inquiries for businesses. Node.js server communicates with Twilio and IBM Watson's intent recognition to automatically generate relevant responses and send them to clients via text message.

TOSbot

Personal Project

A Discord Bot that serves as a moderator for games of Town of Salem (a game similar to Mafia)! Automatically and asynchronously sends prompts to each player via direct message, and collects responses in the form of both messages and reactions. Each day/night cycle, it performs all game calculations efficiently using a priority system.
